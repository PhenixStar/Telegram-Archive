KEY TECHNICAL DETAILS DOCUMENTED
─────────────────────────────────────────────────────────────────────────────

FTS5 Search System
  • File: src/db/fts.py (new 116-line module)
  • Contentless FTS5 virtual table design
  • Query sanitization to prevent injection
  • Batch indexing with crash recovery (checkpoint every 1000 rows)
  • Status tracking via app_settings table
  • Endpoint: GET /api/search?q=query&chat_id=X&limit=50&offset=0
  • Fallback to ILIKE for PostgreSQL/unavailable FTS

Theme System
  • 6 dark themes: Midnight, Telegram Classic, AMOLED Black, Nord, Monokai,
    Solarized Dark
  • 1 light theme: Light Default
  • Auto-detect via prefers-color-scheme media query
  • 4 localStorage keys: tg-theme, tg-theme-auto, tg-theme-dark,
    tg-theme-light
  • CSS custom properties: var(--tg-*)

Keyboard Navigation
  • Arrow keys (⬆/⬇): Browse messages
  • j/k: Navigation shortcuts
  • Space/PageDown: Scroll faster
  • Escape: Close lightbox and modals

Context Menus
  • Messages: Copy text, copy permalink, delete (admin only)
  • Chats: Access chat options
  • Lightbox: Image operations

Message Permalinks
  • Format: #/chat/{chat_id}/msg/{msg_id}
  • Shareable deep links to specific messages
  • Copy link button in context menu
